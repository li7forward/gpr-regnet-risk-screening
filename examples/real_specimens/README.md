# Real Specimen Image Names

Place the real B-scan images here if you want to run the real-specimen scripts.

Expected filenames:

- `foam_line1.png`
- `foam_line2.png`
- `plain_line1.png`
- `plain_line2.png`
- `plastic_line1.png`
- `plastic_line2.png`
- `steel_line1.png`
- `steel_line2.png`

Images are not included in this repository.
